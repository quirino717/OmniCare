# OmniCare Mechanics

O OmniCare foi projetado para operar em ambientes hospitalares, com ênfase em mobilidade omnidirecional, modularidade estrutural e compatibilidade com normas de circulação. 
A base possui arquitetura circular, construída com chapas de MDF e perfis de alumínio, permitindo a integração do sistema elétrico, do manipulador e de um compartimento de carga modular. 
O manipulador foi desenvolvido para interação com elevadores, contando com graus de liberdade translacionais e rotacional no efetuador final.

Nesta seção você encontrará todos os materiais relacionados ao desenvolvimento mecânico do robô, incluindo:

- Modelos CAD completos do robô;
- Estrutura da base omnidirecional;
- Projeto do manipulador;
- Especificações de motores e rodas.

Esses arquivos permitem entender a composição estrutural do robô, seus componentes principais e a lógica de montagem mecânica.
